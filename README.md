Przez ostatnie ~2 lata samodzielnie zaprojektowałem, zbudowałem i utrzymywałem platformę e-commerce **Everkids.pl** — od architektury, przez backend i integracje, po deployment serverless w chmurze.

### Wybrane doświadczenie

#### 🛒 Everkids.pl — platforma e-commerce *(2023–2026)*
Kompleksowa aplikacja zbudowana i wdrożona od zera.
- **Backend:** Node.js / Express na **GCP Cloud Run** (serverless), ~650 endpointów REST
- **Frontend:** React / TypeScript
- **Integracje:** płatności (PayU, Stripe, PayPo) · logistyka (InPost, Apaczka) · marketplace (Allegro) · e-faktury (KSeF)
- **AI:** chatbot oparty o OpenAI API + automatyzacje procesów w n8n
- **Zakres:** architektura, implementacja, deployment, utrzymanie

#### 💻 Freelance — projekty webowe *(Useme)*
Samodzielne prowadzenie zleceń: zbieranie wymagań, wycena, wdrożenie, dokumentacja.
